# What is variable

A variable is like container that stores data.. (a value) 

Exmples:
name = 'raj'
age = ' 19'
height = 5.9

## Rules for variable

✅ must with start "_"(underscore)
✅ Can contain letters , numbers and undersocre (a_z ,A_Z 0_9)
❌ can't start with number --> 123name ,4age
❌ can't use python key words --> class ,def , if ,while,for,yield,async etc ... more


# What is Data Type?

A data type defines the kind of value a variable holds

in python , data types are dynamic , meaning you don't need declare them (python figures it out automatically when you give value).

## common data type

1. numeric Types

        int → whole numbers (10, -5, 1000)

        float → decimal numbers (3.14, -2.7)

        complex → complex numbers (3 + 5j)

2. text Type 

        str -> String("hello","python")


3. Boolean Type

        bool → True or False

4. Sequence Types

        list → ordered, changeable ([1, 2, 3])

        tuple → ordered, unchangeable ((1, 2, 3))

        range → sequence of numbers (range(5) → 0,1,2,3,4)        

5. Mapping Type

        dict → key-value pairs ({"name": "Raj", "age": 21})

6. Set Types

        set → unordered, unique items ({1, 2, 3})

        frozenset → like set but unchangeable

7. None Type

        None → represents nothing or empty value
