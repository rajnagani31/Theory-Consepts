# File FIeld

File Field used for upload multipal data in an applicati like PDF,DOCS,EXCEL,PNG,IMG etc.. Documant

### Fast API Provide UploadFile(method and class) from starlette  

- it's asynchronous object allows you to handle file upload without blocking the event loop. this is perticularly useful when deling large file upload or I/O-bound task (E.G.. saving files to disk or uploading to cloud service)

Note :
Upload file method is designed in Async method but you also use synchronous method if needed

Using async methods allows you to leverage FastAPI's full potential for handling multiple requests concurrently without blocking.
